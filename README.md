# Question 1-Arithmetic Operations

## Problem Statement
This program accepts two integers and calculates their sum, difference, product, and quotient.

## C++ Concepts Practised
Arithmetic operators (+, -, *, /)
Variables
User input/output

## How to Run
Compile and run Q1.cpp, then enter two integers.

## Sample Output
Enter two integers: 10 5
Sum = 15
Difference = 5
Product = 50
Quotient = 2

## Reflection
I learned how arithmetic operators are used to perform calculations on user input.
